# Week 3 Day 1 - Machine Learning Fundamentals

## Objective

The objective of this assignment is to understand the complete Machine Learning workflow using Scikit-learn and Linear Regression. The project demonstrates data exploration, feature selection, train-test splitting, model training, prediction, evaluation, and visualization.

## Dataset

The dataset contains student performance information:

* Study_Hours
* Attendance
* Assignments
* Marks

Marks is the target variable to be predicted.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Plotly
